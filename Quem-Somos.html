O que foi usado para criar esse site!!!

Tecnologias e Recursos Utilizados na Criação do Site Hotel Papper G4

Estrutura e Layout

1. HTML5:
- Utilizado para criar a estrutura básica do site, incluindo a organização dos elementos como navegação, formulários e conteúdo principal. - Utilização de tags semânticas como "nav, header, footer, form," entre outras, para melhorar a acessibilidade e SEO.

2. **CSS3**:
- Utilizado para estilizar o site, definindo cores, fontes, tamanhos e espaçamento entre elementos. - Aplicação de Flexbox para a criação de layouts responsivos e alinhamento de elementos, especialmente para centralizar e justificar botões no formulário. - Implementação de gradientes e personalização de botões para harmonizar com a identidade visual do hotel.

Design Responsivo

**Bootstrap**:
- Utilização do framework Bootstrap para facilitar a criação de um design responsivo e moderno. - Uso de classes de grid e componentes pré-estilizados como navbars, formulários e carrosséis para garantir uma aparência consistente em diferentes dispositivos.

Funcionalidades e Interatividade

1. **JavaScript**:
- Adição de interatividade no site, como a simulação de envio de formulários e exibição de mensagens de confirmação. - Uso de eventos para capturar e processar dados de formulários, incluindo a exibição de resultados de busca.

Conclusão

Com a combinação dessas tecnologias, foi possível criar um site funcional e visualmente atraente para o Hotel Papper G4, proporcionando uma experiência de usuário agradável e intuitiva para reservas e pesquisas. O uso de HTML, CSS, JavaScript, Bootstrap e Flask permitiu a criação de um site dinâmico, responsivo e eficiente.
